
# README

## Requisitos

- Java JDK 8
- NetBeans 12+
- Oracle Database 10g R2
- ojdbc8.jar

## Configuración

1. Abrir el proyecto en NetBeans.
2. Agregar ojdbc8.jar en Libraries.
3. Verificar que Oracle esté activo.
4. No modificar ConexionBD.java.

## Ejecución

- Clean and Build
- Run Project

## Notas

El proyecto fue reorganizado para arquitectura MVC desacoplada.
