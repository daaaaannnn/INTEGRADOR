IMPORTANTE PARA EJECUTAR

1. ESTE PROYECTO DEBE ABRIRSE DESDE:
   NetBeans -> Open Project

2. NO usar "Run File".
   Debes usar:
   Run Project

3. Antes de ejecutar:
   - Click derecho proyecto
   - Clean and Build

4. Agrega la librería:
   ojdbc14.jar

5. Si sale error de clases bloqueadas:
   - Cierra NetBeans
   - Borra manualmente la carpeta build
   - Vuelve a abrir el proyecto

6. Java configurado:
   javac.source=1.4
   javac.target=1.4
