ojdbc8-full.tar.gz - JDBC Thin Driver and Companion JARS
========================================================
This TAR archive (ojdbc8-full.tar.gz) contains the 23.26.0.0.0 release of the Oracle JDBC Thin driver(ojdbc8.jar), the Universal Connection Pool (ucp.jar) and other companion JARs grouped by category. 

(1) ojdbc8.jar (7497817 bytes) - 
(SHA1 Checksum: 163dc3586d18ed4dc7459fdb2b07f350eaa64939)
Oracle JDBC Driver compatible with JDK8, JDK11, JDK12, JDK13, JDK14, and JDK15.

(2) ucp.jar (1498112 bytes) - (SHA1 Checksum: 343040a5a26d8d2ffc50ec7e7ddeea4caa121f33)
Universal Connection Pool classes to be used with ojdbc8.jar -- for performance, scalability, high availability, sharded and multitenant databases.
 
(3) rsi.jar (260581 bytes) - (SHA1 Checksum: 564954f9fe6a03b9e30134b383a44799ba52fe5d)
Reactive Streams Ingestion (RSI) 

======================
Security Related JARs
======================
Java applications require some additional jars to use Oracle Wallets. 
You need to use all the three jars while using Oracle Wallets. 

(4) oraclepki.jar (546295 bytes) - (SHA1 Checksum: 2c27ece80c8804eb1521003288027128dd12874d)
Additional jar required to access Oracle Wallets from Java
