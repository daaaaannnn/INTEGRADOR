# Plan comercial resumido

## Propuesta de valor

El sistema centraliza matrícula, control de horas, evaluación, evidencias, trazabilidad y certificados de prácticas académicas. Está pensado para instituciones que requieren seguimiento normativo, reportes para acreditación y evidencia verificable.

## Módulos vendibles

1. Gestión de usuarios y roles.
2. Gestión de prácticas, grupos e instituciones.
3. Control de horas y confirmación institucional.
4. Evaluación con rúbricas parametrizables.
5. Gestión documental.
6. Trazabilidad y auditoría.
7. Reportes PDF y certificados con hash.

## Modelo sugerido de comercialización

- Licencia local anual por sede.
- Parametrización inicial de base de datos Oracle.
- Capacitación de 2 horas para usuarios finales.
- Soporte mensual para ajustes de reportes y respaldos.

## Diferenciadores

- Auditoría sobre operaciones críticas.
- Compatible con Oracle 10g+ y NetBeans.
- Documentación incluida desde el primer día.
- Código organizado por capas para mantenimiento futuro.
