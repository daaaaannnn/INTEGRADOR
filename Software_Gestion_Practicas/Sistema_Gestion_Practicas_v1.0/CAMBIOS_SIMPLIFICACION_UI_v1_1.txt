CAMBIOS DE SIMPLIFICACIÓN v1.1
================================

Se añadieron pantallas nuevas para evitar que el usuario escriba ID manuales y para reducir errores ORA-02291, ORA-01722 y ORA-01861.

1. Coordinador
- Vincular Estudiantes ahora abre una pantalla con listas desplegables de Estudiante, Práctica, Institución y Grupo.
- Si no existe grupo, se puede crear desde la misma pantalla.
- La tabla muestra el avance de matrícula, horas reportadas, horas aprobadas y promedio sin mostrar ID al usuario.

2. Docente
- Banco de Preguntas y Evaluar Estudiante ahora abren una pantalla simplificada.
- El docente selecciona una matrícula real y publica preguntas.
- Las preguntas quedan asociadas a una evaluación real y el estudiante puede verlas desde su panel.
- El docente puede registrar nota y observación final.

3. Estudiante
- Registrar Actividad/Horas abre el control histórico de horas.
- Responder Cuestionario muestra solamente las preguntas asociadas a sus matrículas.
- Consultar Calificación muestra estado de evaluación, respuesta y nota.

4. Institución
- Confirmar Horas y Estudiantes Asignados abren el histórico de horas.
- La institución puede aprobar o rechazar horas reportadas.

5. Director / Administrador
- Reportes ahora abre un panel de histórico con resumen general, horas, evaluaciones, respuestas, LOG_ACTIVIDAD, HISTORIAL_REVISION y reportes generados.

6. Corrección técnica
- Se corrigió PreguntaDAO para no consultar EVALUACION.TITULO, porque la tabla EVALUACION no tiene esa columna.
- Las nuevas pantallas usan PRACTICA.TITULO, no PRACTICA.NOMBRE.
- Las fechas se manejan como java.sql.Date y aceptan yyyy-MM-dd o dd/MM/yyyy.
- Se registra trazabilidad en LOG_ACTIVIDAD e HISTORIAL_REVISION cuando aplica.

Recomendación de uso:
1) Coordinador: crear/seleccionar grupo y matricular estudiante.
2) Docente: publicar evaluación con preguntas.
3) Estudiante: responder cuestionario y registrar horas.
4) Docente/Coordinador/Institución: validar horas.
5) Director: revisar histórico y reportes.
