SOLUCION RAPIDA - PROYECTOJD

1. Error Unable to delete directory build/classes/.../dao
   No es error del codigo. Es bloqueo de Windows, OneDrive, NetBeans o java.exe.
   Solucion:
   - Cierre el programa si quedo abierto.
   - Cierre NetBeans.
   - Ejecute limpiar_build_windows.bat dentro de la carpeta del proyecto.
   - Abra NetBeans y haga Clean and Build.
   Recomendacion: trabajar fuera de OneDrive, por ejemplo C:\Proyecto\GestionPracticas.

2. Error The Network Adapter could not establish the connection
   La app no podia conectar porque estaba probando XE y su conexion de SQL Plus usa orcl.
   Se actualizo ConexionBD.java para intentar primero:
   - jdbc:oracle:thin:@localhost:1521:orcl
   - jdbc:oracle:thin:@127.0.0.1:1521:orcl
   - ORCL
   - XE

3. No requiere cambios en tablas de la base de datos.
   Solo verifica que OracleServiceORCL y OracleOraDb...TNSListener esten iniciados.

4. Prueba en SQL Plus:
   conn proyectojd/proyectojd@orcl
   select * from tab;
