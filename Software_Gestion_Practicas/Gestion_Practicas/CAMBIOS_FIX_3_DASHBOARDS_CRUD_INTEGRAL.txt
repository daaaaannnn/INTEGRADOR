FIX 3 - Dashboards por rol, CRUD genérico y datos base de prueba

Cambios realizados:
1. Dashboard Estudiante mejorado:
   - Panel completo con menú lateral.
   - Accesos a Registrar Actividad/Horas, Cuestionario, Reportes y Consulta de Evaluación.
   - Botón Crear Matrícula Prueba para generar datos mínimos si el estudiante no tiene práctica asignada.

2. Dashboard Docente mejorado:
   - Accesos a Rúbricas, Banco de Preguntas, Evaluar Estudiante, Revisar Actividades y Respuestas/Feedback.
   - Incluye resumen de evaluaciones, rúbricas, preguntas y actividades.

3. Dashboard Coordinador creado de forma completa:
   - Gestión de grupos, asignación docente, matrícula de estudiantes, instituciones receptoras y control de horas.
   - Incluye botón Crear Datos Base.

4. Dashboard Institución mejorado:
   - Confirmación de horas, estudiantes asignados, prácticas en sede y datos de institución.
   - Incluye institución receptora base para pruebas.

5. Se agregó CrudTablaFrame:
   - Ventana genérica para listar, guardar, editar y eliminar/inactivar registros.
   - Detecta columnas existentes para reducir ORA-00904 cuando la BD de la U tiene nombres diferentes.

6. Se agregó DBHelper y SemillasProyecto:
   - DBHelper consulta columnas reales, cuenta registros, inserta y actualiza dinámicamente.
   - SemillasProyecto crea datos base mínimos: institución, programa, curso, grupo, práctica, matrícula y rúbrica.

7. Script nuevo:
   - database/06_datos_base_para_pruebas_universidad.sql
   - No borra tablas. Solo deja usuarios mínimos por rol con contraseña 1234.

Validación:
- Compilado con Ant.
- Resultado: BUILD SUCCESSFUL.

Importante:
- No se pudo probar contra la red Oracle real de la universidad desde este entorno.
- Si aparece un ORA-00904 adicional, enviar DESC de la tabla afectada para ajustar ese DAO o usar el CRUD genérico incluido.
