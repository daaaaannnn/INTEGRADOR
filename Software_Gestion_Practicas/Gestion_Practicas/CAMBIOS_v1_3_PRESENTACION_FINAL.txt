Cambios v1.3 presentación final

1. Reportes e histórico
- Se agregó descarga de PDF para cualquier vista cargada.
- Se agregó botón PDF comentarios, que carga evaluaciones, respuestas y comentarios docentes y genera el PDF.
- Se agregó CRUD básico sobre REPORTE: modificar título/comentario y eliminar/inactivar reporte.
- Se muestran actividades del estudiante y horas desde REGISTRO_ACTIVIDAD, además de horas aprobadas.

2. Evaluación docente
- Se agregó ventana emergente para ver preguntas, respuestas del estudiante y retroalimentación.
- El docente puede guardar un comentario visible para el estudiante.
- Se agregó opción para ajustar etapa académica, estado de matrícula y horas requeridas de la práctica.
- Se agregó eliminación/inactivación de evaluación.

3. Estudiante
- El cuestionario muestra estado de evaluación, estado de respuesta, si está pendiente o finalizada, nota y comentario docente.
- Si la evaluación ya está finalizada, el estudiante no puede modificar la respuesta.

4. Control de horas
- El registro ya no depende exclusivamente de HORAS_PRACTICA. Registra en REGISTRO_ACTIVIDAD y, si es compatible, también en HORAS_PRACTICA.
- Histórico visible de actividades, horas reportadas, aprobadas, estado, actividad y observación.
- Docente, coordinador o institución pueden aprobar, rechazar, actualizar observación o eliminar/inactivar registros.

5. Coordinador e Institución
- Se agregó Seguimiento integral: grupo -> estudiantes -> práctica -> institución -> docente -> horas -> evaluaciones.
- Grupos de práctica, asignación docente y estudiantes asignados abren la vista integral en lugar de formularios con IDs.

6. SQL adicional
- Se incluye sql/99_patch_presentacion_v13.sql para agregar columnas faltantes sin borrar datos.
